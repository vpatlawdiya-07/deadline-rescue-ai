# 📖 Project Overview

AI Deadline Rescue is a smart productivity web application designed to help students and professionals stay organized.

Users can:

* Add tasks
* Set priorities
* Manage deadlines
* Track completion status
* Generate AI-powered productivity plans using Gemini AI

The application combines Firebase services with Google's Gemini API to provide intelligent task management and personalized suggestions.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/vpatlawdiya-07/deadline-rescue-ai.git
```

Move into the project

```bash
cd deadline-rescue-ai
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

# 🎯 Problem Statement

Managing multiple assignments, projects, and deadlines can be overwhelming for students and professionals.

Traditional to-do list applications simply store tasks but do not provide intelligent guidance.

AI Deadline Rescue solves this problem by integrating Google's Gemini AI to analyze deadlines and generate personalized productivity plans.

The application combines:

* Task management

to create a smart productivity assistant that helps users prioritize work and stay on schedule.
